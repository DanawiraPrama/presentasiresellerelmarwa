Google Form

Input Data Terstruktur untuk Reseller

Keunggulan Google Form

  • Gratis dan mudah digunakan
  • Terintegrasi langsung dengan Google Spreadsheet
  • Bisa diakses via HP tanpa instalasi
  • Mendukung berbagai jenis input (teks, pilihan, upload)
  • Notifikasi otomatis saat ada pendaftaran baru

Contoh form pendaftaran reseller menggunakan Google Form

Integrasi dengan Spreadsheet

  • Semua data form otomatis masuk ke Google Sheet
  • Admin bisa verifikasi data langsung dari spreadsheet
  • Kolom khusus untuk status verifikasi (pending/diterima)
  • Filter data berdasarkan wilayah dan status

WA Broadcast Tools

Otomasi Pengiriman Massal Gratis

Workflow Broadcast

1
Ekspor kontak reseller

Dari Google Spreadsheet ke file CSV

2
Import ke WA Sender

Pilih grup atau kontak individual

3
Buat template pesan

Dengan variabel personalisasi (nama, dll)

4
Jadwalkan pengiriman

Atur waktu optimal (pagi/sore)

Tips Anti-Banned

  • Batasi 30-50 pesan/jam per nomor
  • Gunakan delay 1-2 menit antar pesan
  • Variasi konten - jangan spam
  • Gunakan nomor berbeda untuk broadcast dan support

Google Drive Management

Pusat Materi Promosi Terorganisir

Struktur Folder

1
Folder Utama "El Marwa Reseller"

Akses untuk semua admin

2
Subfolder "Materi Promosi"

Brosur, video, template caption

3
Subfolder "Per Angkatan"

Batch 1, Batch 2, dst

4
Subfolder "Training"

Rekaman webinar, panduan PDF

Cara Berbagi Materi

  • Set folder ke "Anyone with link can view"
  • Gunakan link pendek (bit.ly) untuk mempermudah
  • Buat QR code untuk akses cepat
  • Update versi file tanpa mengubah link
